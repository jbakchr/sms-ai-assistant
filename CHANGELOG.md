# Changelog

All notable changes to this project will be documented in this file.

This project follows a pragmatic, pre-1.0 versioning scheme.
Breaking changes may occur between minor versions.

---

## v0.2.0 – First usable prototype

### Added

- Chrome extension integration with Google Messages (web)
- UI button to request AI-generated SMS replies
- Automatic insertion of suggested replies into the message input field
- Local FastAPI backend for request orchestration
- Integration with Ollama Cloud models for high-quality Danish replies

### Changed

- Project documentation expanded with detailed Getting Started guides
- Backend and Chrome extension responsibilities clearly separated

### Notes

- This is an early, developer-focused prototype
- Replies are suggestions only and are never sent automatically

---

## v0.1.0 – Initial release

- Initial project setup
- README describing the project vision and architecture
- Defined a local, privacy-first approach for AI-generated SMS replies
- Chrome extension + local backend architecture outlined
